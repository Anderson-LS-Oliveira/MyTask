# MyTask

Sistema web simples de gerenciamento de tarefas desenvolvido para a disciplina de Fábrica de Software I da Universidade da Região de Joinville.

Repositório:
https://github.com/Anderson-LS-Oliveira/MyTask.git

---

## Sobre

O MyTask é uma aplicação que permite ao usuário cadastrar, visualizar e acompanhar tarefas do dia a dia de forma prática. O sistema foi criado com o objetivo de auxiliar na organização pessoal, facilitando o controle e o acompanhamento das atividades.

---

## Funcionalidades

* Cadastro de tarefas com título e descrição
* Listagem de tarefas
* Marcação de tarefas como concluídas
* Visualização da descrição
* Armazenamento local com LocalStorage

---

## Tecnologias

HTML, CSS, JavaScript e Bootstrap.

---

Projeto acadêmico do curso de Engenharia de Software, desenvolvido como atividade avaliativa.
